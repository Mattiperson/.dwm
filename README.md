## Installation
How to install this and dependancies.
```
git clone https://github.com/Mattiperson/.dwm

cd .dwm

sudo make clean install

cd

mkdir .config

cd .config

git clone https://github.com/Mattiperson/slstatus

cd slstatus

sudo make clean install

cd

```
